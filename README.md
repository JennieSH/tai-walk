# 台灣走走 - Tai Walk

台灣旅遊景點導覽，串接交通部的 [TDX API](https://ptx.transportdata.tw/MOTC/) ，提供台灣景點、活動和餐廳查詢。

![Tai Walk ](https://i.imgur.com/Xye6bjM.jpg)



## Demo

🚀 [https://jenniesh.github.io/tai-walk](https://jenniesh.github.io/tai-walk)

## Features

- [x] RWD
- [x] 景點、活動和餐廳三大分類查詢
- [x] 支援**關鍵字**、**日期**、**縣市**和**經緯度**查詢
- [x] Google Maps 地圖
